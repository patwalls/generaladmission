# API Endpoints

## HTML API

### Users

- `GET /users/new`
- `POST /users`
- `PATCH /users`

### Session

- `GET /session/new`
- `POST /session`
- `DELETE /session`

## JSON API

### Friends

- `POST /api/friends`
- `GET /api/friends` (index)
- `DELETE /api/friends/:id`

### Artists

- `POST /api/artists`
- `GET /api/artists/:id`
- `PATCH /api/artists/:id`
- `DELETE /api/artists/:id` (not sure if I need this)

### Attends

- `GET /api/attends`
- `POST /api/attends`
- `GET /api/attends/:id`
- `PATCH /api/attends/:id`
- `DELETE /api/attends/:id`
